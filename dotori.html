DOTORI

흩어진 디자인 프로젝트를 다시 이어주는 파일 정리 에이전트

도토리는 뒤섞인 디자인 프로젝트 파일을 분석하고,
작업 과정과 최종 결과물을 다시 찾기 쉽게 정리해주는 에이전트입니다.

WHERE IS MY FINAL?

최종.psd, 진짜최종.psd, 최종수정2.psd처럼
프로젝트가 끝난 뒤에도 수많은 파일이 남습니다.
시간이 지나면 어떤 파일이 최종본인지,
어떤 순서로 작업했는지 찾기 어려워집니다.

DOTORI DOES

  • 프로젝트 폴더 전체 분석
  • 작업 파일과 최종 결과물 구분
  • 파일의 버전 관계와 작업 흐름 복원
  • 판단이 어려운 파일은 사용자에게 확인
  • 원본을 건드리지 않고 복사본으로 안전하게 정리
  • 포트폴리오용 과정 이미지와 최종 이미지 추천

HOW IT WORKS

  1. 프로젝트 폴더 전달
  2. 파일 분석
  3. 작업 흐름 추정
  4. 필요한 질문
  5. 정리 계획 미리보기
  6. 사용자 승인
  7. 정리본 생성

파일을 정리하는 것이 아니라,
프로젝트를 기억합니다.
